- var 은 재선언이 가능하지만 let, const 는 재선언이 불가하다.
- let 은 재선언은 불가하지만 재할당은 가능하다.
- const 는 재선언도 재할당도 불가하다. (객체라면 재할당은 가능하다.)

- 변수의 선언은 var, let, const로 할 수 있다.
- 자바스크립트에서 변수 선언은, 선언 → 초기화 단계를 거쳐서 수행된다.
- 자바스크립트 엔진은 소스코드를 한 줄씩 순차적으로 실행하기에 앞서, 변수 선언을 포함한 모든 선언문(ex. 변수 선언문, 함수 선언문 등)을 찾아내 먼저 실행하는데, 즉 변수 선언이 어디에 있든 상관없이 다른 코드보다 먼저 실행되는 특징을 호이스팅(hoisting)이라 한다.
