- Node.js는 싱글 스레드이다.
- 자바스크립트 엔진은 Memory Heap과 Call stack으로 이뤄져 있다.
- Memory Heap은 메모리 할당이 일어나는 곳이고, Call Stack은 코드 실행에 따라 호출 스택이 쌓이는 곳이다.
- 콜스택에 쌓인 실행 컨텍스트(작업)에 따라 위에서부터 차례로 실행이 일어나는 곳이기 때문에 비동기 처리를 할 수 없다.
- 하지만 이벤트루프가 자바스크립트 엔진으로부터 넘겨받은 비동기 작업을 스레드 풀에 할당해서 작업들을 비동기로 처리 할 수 있게 해준다.

- 즉, 이벤트루프는 이벤트큐에 쌓여있는 작업들을 하나하나씩 동기적으로 처리하는것이 아닌, 이벤트큐에 쌓여있는 작업들을 쓰레드 풀로 전달해 비동기적으로 작업을 처리하게 된다.
