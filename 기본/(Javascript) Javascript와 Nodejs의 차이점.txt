- JavaScript는 독립적인 언어가 아니라 스크립트 언어로써, 특정한 프로그램 안에서만 동작을 할 수 있었는데 웹 브라우저가 없으면 사용할 수 없다는 단점이 있었다. 그에따라 client 개발을 위한 용도로만 국한되어 있었다.
- JavaScript를 크롬(Chrome)과 같은 브라우저에서만 쓰는 것이 아닌 브라우저 밖 즉, 내 컴퓨터에서 다양한 용도로 확장하기 위해 만들어진 것이 바로 Node.js이다.

- Node.js 는 Chrome V8 Javascript 엔진으로 빌드된 Javascript 런타임이다.
- 런타임은 프로그램이 실행되고 있을 때 존재하는 곳 즉 프로그래밍 언어가 구동되는 환경이라고 이해하면 된다.

++++++++++++++++++++++

- JavaScript는 프로그래밍 언어이다.
- JavaScript는 브라우저 없이는 사용할 수 없었다.
- 따라서 브라우저 밖에서도 사용 할 수 있도록 해보자 라는 생각에 만들어낸게 Node.js다.
- 즉 Node.js는 Javascript 런타임이다.
