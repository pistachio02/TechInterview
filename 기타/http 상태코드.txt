- HTTP 상태 코드는 클라이언트가 보낸 HTTP 요청에 대한 서버의 응답 코드로, 상태 코드에 따라 요청의 성공/실패 여부를 판단하게 된다.

- 1xx : 요청을 받았으며 작업을 계속함 을 의미한다.
- 2xx : 요청을 성공적으로 받았으며, 인식했고, 수용함 을 의미한다.
- 3xx : 요청 완료를 위해 추가 작업 조치가 필요함 을 의미한다.
- 4xx : 요청의 문법이 잘못되었거나 요청을 처리할 수 없음 을 의미한다.
- 5xx : 서버가 명백히 유효한 요청에 대해 충족을 실패했음 을 의미한다.
    ㄴ (The server failed to fulfill an apparently valid request.)
